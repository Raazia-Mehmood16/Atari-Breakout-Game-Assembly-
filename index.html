🎮 Atari Breakout

Assembly Language implementation of the classic Atari Breakout game. Built using x86 Assembly with paddle movement, ball physics, collision detection, scoring system, lives system, sound effects, and win/lose screens.

Project Overview

This project recreates the famous Atari Breakout game completely in Assembly Language. Players control a paddle using the arrow keys and must destroy all bricks using a bouncing ball.


The game includes:

  • ✔ 40 Bricks
  • ✔ Score Tracking
  • ✔ 3 Lives System
  • ✔ Sound Effects
  • ✔ Paddle Collision Detection
  • ✔ Brick Destruction Logic
  • ✔ Win & Game Over Screens

Submitted by: Raazia Mehmood (24F-0614) & Sabeeka (24F-0654)

Game Features

🏓 Paddle Controls

Move paddle smoothly using Left and Right Arrow keys.

⚽ Ball Physics

Ball bounces realistically from walls, paddle and bricks.

🧱 Brick System

40 colorful bricks arranged in rows with collision detection.

🔊 Sound Effects

Different sounds for brick hits, paddle hits and life loss.

❤️ Lives System

Player starts with 3 lives before game over.

🏆 Score Tracking

Gain points for every brick destroyed.
